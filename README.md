# 📅 Meetup Lite

A lightweight event and meetup management platform built with the MERN stack. It enables users to create, join, and track meetups with an intuitive interface, streamlined user experience, and real-time updates.

---

## 🌐 Tech Stack

**Frontend:** React.js, Tailwind CSS, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT, Bcrypt.js  
**Other Tools:** React Router DOM, Context API

---

## 📌 Features

- 🔐 **User Authentication**
  - Secure login & signup using JWT
  - Role-based access for organizers & attendees

- 📅 **Event Management**
  - Create, update, delete meetups
  - Categorize events by type and tags

- 🙋 **Join/Leave Events**
  - One-click RSVP and cancel participation
  - Real-time attendee list updates

- 📊 **Event Overview**
  - Dashboard to view upcoming, ongoing, and past events
  - Track number of attendees, dates, and status

- 🧑‍🤝‍🧑 **User Engagement**
  - See who’s attending
  - View details of each meetup like description, date, and host info
